WebGL Pipeline

  1. Initialize WebGL
  2. Create Arrays (geometry information)
  3. Vertex Buffer and Index Buffer (use previous arrays)
  4. Create, compile and link the Shaders
  5. Create attributes and associate them with buffer objects using JavaScript.
  6. Create uniforms and associate them using JavaScript.
  7. Create Transformation Matrix using JavaScript.

Steps:

1. Vertex Shader

  • We invoke "drawElements" and "drawArray" methods.
  • The vertex shader is executed for each vertex provided in the vertex buffer object.
  • It calculates the position of each vertex of a primitive polygon.
  • It stores it in the varying gl_position.
  • It calculates other attributes asociated with a vertex (color, texture coords, vertices)

2. Primitive Assembly

  • The triangles are assembled and passed to the rasterizer.

3. Rasterization

  • The pixels in the final image of the primitive are determined.
  • 1. Culling: Process to discard triangle with improper orientation.
  • 3. Clipping: Clips triangle outside the view area.

4. Fragment Shader

  • Gets varyings and primitives from the rasterization phase.
  • Calculates the color values for each pixel between the vertices.

5. Fragment Operations

  • Depth, color buffer blend, dithering.

6. Frame Buffer

  • Final destination.
  • Portion of graphics memory that hold the scene data.
